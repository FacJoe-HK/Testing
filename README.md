# Modern Clocking App

A minimal, always-on clock with timezone awareness and a glassy modern UI.

## Features
- Live digital clock with seconds
- 12h/24h toggle
- Quick timezone switcher (HK, UTC, London, New York, Tokyo)
- Clean, desk-friendly design with glassmorphism styling

## Getting started
```bash
npm install
npm run dev -- --host --port 4173
```
Then open the forwarded Codespaces URL for port **4173**.

## Build
```bash
npm run build
```
